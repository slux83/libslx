# libslx
C++ Advanced library with a lot of features:

- Core
  - Application processing
  - Signal/Slots
  - Varint 
  - Timers
  - Logging
- Serialization/Deserialization
- Network
  - TCP Sockets (client/server)
  - UDP Socktes (client/server) 
- Concurrency
  - Thread pools
  - Mutex
  - Semaphore
  - Threads
  - Blocking queues
  - Waiting conditions
